# ke2daira

## まさかり　くさお

苗字と名前の最初の文字を入れ替えてみました(Fortranは英数字入力のみ対応)

### 手順(Fortran)
1. ./ke2daira_F を実行

2. 1フレーズ目を入力(例：kusakari)

3. 2フレーズ目を入力(例：masao)

4. 1フレーズ目を何文字目から入れ替えるか入力(例：1)

5. 2フレーズ目を何文字目から入れ替えるか入力(例：1)

6. 1フレーズ目を何文字目まで入れ替えるか入力(例：2)

7. 1フレーズ目を何文字目まで入れ替えるか入力(例：2)

8. 出力されます(例：masakari kusao)
